# Dotfiles
This is my personal linux config

## Prerequisite

* **WM:** bspwm
* **Bar:** polybar
* **Launcher:** rofi
* **Compositor:** picom (ibhagwan's build)
* **Notification Daemon:** dunst
* **Terminal:** urxvt, urxvt-perls (for kb shortcuts)
* **Sysmon:** htop
